UI Layout Types

Full Screen

Screens that need no UI mount directly in index.html.

Global Navigation

Mounting to this template adds the nav-menu and scroll-away nav-bar

Main Menu

A left hand menu that opens with the global menu icon, in the nav-bar.

Module Navigation

Adds a sticky header beneath the nav-bar, and a collapsible icon-bar to the left.

Subscreen Navigation

Deep screens replace the icon-bar with a back arrow.

Subscreen Tabs

Tabs convert to a select menu on smaller screens.
